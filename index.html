Sobre

O site organiza e exibe para o usuário os filmes que são lançamentos, filmes em destaque e suas últimas avaliações, entre outras coisas, baseado no popular site de filmes IMDB(Internet Movie Database).

Este site foi desenvolvido por um aluno do primeiro semestre de Sistemas de Informação na PUC Minas, para a disciplina de Desenvolvimento de Interfaces Web. Utilizando de HTML5, CSS e JavaScript, além de implementar o Bootstrap Framework na versão 4.
